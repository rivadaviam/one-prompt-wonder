# Quickstart: Banh Mi Vietnam Replica

## Prerequisites

- Node.js available locally
- npm available locally

## Run Locally

From the repository root:

```sh
cd frontend
npm install
npm run dev
```

Open the local Vite URL shown in the terminal.

## Manual Validation

1. Confirm the first viewport shows the Banh Mi Vietnam hero and navigation for Top, Story, Anatomy, Fillings, and Street icon.
2. Select each navigation item and confirm it scrolls to the matching page section.
3. Confirm the story timeline includes The arrival, The rebirth, and Global recognition.
4. Confirm the anatomy section shows ingredient labels for baguette, cold cuts, pork rolls, margarine, sauce, pate, pickled daikon, cucumber, coriander, pepper, carrot, and chilli.
5. Confirm the fillings section communicates varied fillings and Vietnamese dish pairings.
6. Confirm the footer includes the street availability message and copyright/creator line.
7. Check one mobile-width viewport and one desktop-width viewport for readable text, usable navigation, and no incoherent overlap.

## Build Check

```sh
cd frontend
npm run build
```
