# Bánh Mì Vietnam

Frontend-only recreation of the supplied Bánh Mì Vietnam reference, built in
the existing Vite harness. All imagery, fonts, icons, and animation libraries
needed by the experience are stored locally.

## Stack

- Vite
- JavaScript
- Plain CSS

## Commands

```bash
npm install
npm run dev
npm run build
```

## Local Demo

Run:

```bash
npm run dev
```

Then open the local URL printed by Vite.

The production build is generated in `dist/` with `npm run build`.
